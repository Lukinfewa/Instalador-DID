# Instalador DID

Aplicación de escritorio de lista de tareas creada con React, Vite y Electron.

## Instalar dependencias

```powershell
npm install
```

## Ejecutar en desarrollo

```powershell
npm run dev:electron
```

## Generar el instalador

```powershell
npm run dist
```

El instalador `.exe` se genera localmente en la carpeta `release/`.
